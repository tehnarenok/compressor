# Архиватор

## Как работать с ним

Архивация

```bash
compressor compress -f <исходный файл> -o <куда сохранить>
```

Деархивация

```bash
compressor decompress -f <сжатый файл> -o <куда сохранить>
```

## Что внутри:

- Преобразование Барроуза-Уилтера
- Преобразование методом стопки кнги
- Хаффман
